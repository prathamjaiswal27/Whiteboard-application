# Whiteboard Frontend

A React whiteboard app with drawing tools, a toolbar, and adjustable toolbox settings.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## Scripts

- `npm start` runs the app in development mode.
- `npm run build` creates a production build in the `build` folder.
- `npm test` starts the test runner.

## Notes

This project was bootstrapped with Create React App. If `npm start` cannot find `react-scripts`, run `npm install` first.
